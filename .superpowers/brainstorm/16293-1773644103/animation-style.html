Animated playfield — what does it do?

All three use the frosted gem circles and active palette. Pick the motion that fits.

A — Auto-play

The grid plays itself. A random 3×3 block rotates every second or so using the selected transition animation. Looks like a game in progress.

B — Colour wave

A slow ripple of colour cycles diagonally across the grid. Each tile shifts to the next palette colour with a staggered delay based on its distance from the origin.

C — Idle shimmer

Individual tiles randomly scale up and glow, then settle back. Calm ambient motion — grid looks alive but not busy.
